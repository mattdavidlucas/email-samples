Intel Icons: Landing Page Usage

For standard, 26px by 26px blue icons, use the class names below. Additionally, always use the class name icon-sm as well.

Example:

<h3 class="icon-sm blue-chip-sm"></h3>

For large, 36px by 36px blue icons, change -sm to -lg.

Example:

<h4 class="blue-chip-lg"></h4>

For white icons, change blue to white and -sm to -lg.

Example:

<h4 class="white-chip-lg"></h4>
